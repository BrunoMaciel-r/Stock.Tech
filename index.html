Artigos de suporte

1. COMO ADICIONAR UM NOVO PRODUTO AO ESTOQUE?

Para adicionar um produto, acesse o menu Produtos.

Siga os passos abaixo:

  • Acesse Produtos
  • Clique em Adicionar Produto
  • Preencha nome, preço e quantidade
  • Clique em salvar

2. COMO EDITAR AS INFORMAÇÕES DE UM PRODUTO?

Localize o item desejado na lista de produtos.

  • Abra o menu Produtos
  • Encontre o produto
  • Clique em editar
  • Atualize as informações
  • Salve as alterações

3. POR QUE NÃO APARECEM INFORMAÇÕES NO DASHBOARD?

Isso pode acontecer quando não existem produtos cadastrados.

  • Cadastre produtos
  • Atualize a página
  • Verifique sua conexão
  • Confira o banco de dados

4. COMO REMOVER UM PRODUTO DO ESTOQUE?

Saiba como excluir produtos cadastrados do sistema.

  • Acesse o menu Produtos
  • Encontre o item desejado
  • Clique em excluir
  • Confirme a remoção

5. COMO FUNCIONA O CONTROLE DE ESTOQUE?

Entenda como o sistema controla entradas e saídas.

  • Entradas aumentam o estoque
  • Saídas diminuem a quantidade
  • O dashboard atualiza automaticamente
  • Os relatórios são gerados em tempo real

6. COMO ALTERAR O TEMA DO SISTEMA?

Aprenda como ativar o modo escuro.

  • Clique na engrenagem no topo
  • Acesse Aparência
  • Escolha Dark Mode
  • Salve as alterações
